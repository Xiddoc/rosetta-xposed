rosetta-xposed — API documentation

This -javadoc artifact is intentionally a placeholder. The published library
ships an empty javadoc jar (Maven Central requires the artifact to exist, not
to contain rendered docs); the human-facing documentation lives at:

  https://xiddoc.github.io/rosetta-xposed/

Generating real API docs requires the Dokka Gradle plugin, which is kept out of
the default offline build (it would need network resolution + dependency-
verification pins). See docs/reference/building.md ("Publishing").
